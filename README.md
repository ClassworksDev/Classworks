# Classworks

Classworks的前端代码

## 📦 安装和运行

1. 克隆项目

```bash
git clone https://github.com:ClassworksDev/Classworks.git
cd Classworks
```

2. 安装依赖

```bash
pnpm install
```

3. 运行开发服务器

```bash
pnpm run dev
```

4. 构建生产版本

```bash
pnpm run build
```
