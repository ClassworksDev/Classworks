基于 Vue 3 + Vuetify 的现代化作业管理系统前端项目

## ✨ 特性

- 🎯 TypeScript 支持
- 🎨 基于 Vuetify 3 的精美 UI
- 📱 响应式设计，完美适配多端
- ⚡️ Vite 提供的极速开发体验
- 🔑 完善的权限管理系统
- 🎉 丰富的组件和功能模块

## 📦 快速开始

### 环境准备

- Node.js 16+
- pnpm 7+

### 安装步骤

```bash
# 克隆项目
git clone https://github.com:ClassworksDev/Classworks.git
cd Classworks

# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```

## 📂 项目结构

```
Classworks/
├── src/           # 源代码
├── public/        # 静态资源
├── components/    # 通用组件
├── views/         # 页面
├── store/         # 状态管理
└── utils/         # 工具函数
```
